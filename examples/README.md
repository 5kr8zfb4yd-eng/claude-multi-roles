# Examples

Worked, copy-pasteable walkthroughs of the framework in motion.

- [**uiux-to-swe-handoff**](./uiux-to-swe-handoff/) — a UI/UX session builds a
  component against mock data and hands the data contract to SWE through `BUS.md`.
  Shows the exact artifacts each session writes.

Want to contribute one? Add a folder here with a short `README.md` plus the real
`BUS.md` / `MEMORY.md` excerpts it produces, and link it above.
